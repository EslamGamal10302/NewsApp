
# Your Meal 

Native Mobile Android Application to set weekly food plan, open source and free,using design architecture MVP, java rx, room , firestore.





## Application Features :

- Login and Register

- Daily Inspirations

- meals planned for today

- search by all Meals

- Search by areas

- search by ingrediants

- search by category

- user can save meals

- add meals to week plan

- user can join as guest

- May work locally

## Permissions
- Network Permission.
## Tools
- TheMealDB API

- RX java

- Room

- Retrofit

- Firebase

- Lottie animations

- MVP Architectural Pattern
